Project 1

Real Estate Club GVSU Website

The Real Estate Club GVSU Website is a comprehensive platform designed to streamline the management and dissemination of information for the Real Estate Club at Grand Valley State University.

Overview:

Frontend: The website is built using React, which offers a dynamic and responsive user interface, ensuring an optimal user experience across various devices.

Backend: The backend leverages Node.js and Firestore, providing a robust and scalable environment to handle data and user authentication seamlessly.

API Integration: The site utilizes API calls to fetch and update information dynamically, ensuring that the content remains current and relevant.

Features:

  • Admin Access:
    • Admin access is secured through a specific password. This ensures that only authorized club members can make changes to the site content.
    • The admin dashboard allows users to update announcements, calendar events, and other essential information without requiring any programming skills.
  • User-Friendly Interface:
    • The frontend interface, developed in React, offers an intuitive and interactive experience for users.
    • The design prioritizes accessibility and ease of use, enabling club members to navigate and utilize the website efficiently.
  • Real-Time Updates:
    • By integrating Firestore, the website supports real-time updates. Any changes made through the admin dashboard are instantly reflected across the site.
    • This ensures that all members have access to the latest information at all times.
  • Security:
    • The use of Node.js in the backend allows for secure handling of user data and authentication processes.
    • The system is designed to protect against unauthorized access and ensure data integrity.
  • Collaboration and Communication:
    • The platform facilitates better communication among club members by providing up-to-date information on events, meetings, and announcements.
    • The integration of social media links further enhances the clubâ€™s online presence and engagement.

Technologies Used:

  • React: For building a dynamic and responsive frontend.
  • Node.js: To handle the backend logic and server-side operations.
  • Firestore: For real-time database management and data storage.
  • Tailwind CSS: To ensure a modern and consistent design throughout the site.

Role:

As a key contributor to this project, my responsibilities included:

  • Developing the React frontend to create an interactive and user-friendly interface.
  • Setting up the Node.js backend to manage API calls and data handling.
  • Integrating Firestore for efficient data storage and real-time updates.
  • Implementing the admin access feature, allowing secure and easy updates to site content by club members.
  • Ensuring the overall design and functionality meet the club's requirements and provide an optimal user experience.
